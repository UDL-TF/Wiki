# Request Changes

Don't have a GitHub account or not comfortable with Git? You can still contribute to the wiki by submitting change requests!

---

## How to Request Changes

### Option 1: GitHub Issues (Recommended)

The best way to request changes is through GitHub Issues:

1. Go to [Wiki Issues](https://github.com/UDL-TF/Wiki/issues)
2. Click **"New Issue"**
3. Choose the appropriate template:
   - **Content Correction** - Fix incorrect information
   - **New Content Request** - Suggest new pages or sections
   - **General Feedback** - Other suggestions

4. Fill in the details and submit

!!! info "GitHub Account Required"
    You'll need a [free GitHub account](https://github.com/signup) to submit issues.

---

### Option 2: Discord

Discuss changes with the community on Discord:

1. Join the [UDL Discord](https://udl.tf)
2. Find the wiki or feedback channel
3. Describe your suggestion

A contributor will help submit the change on your behalf.

---

## What to Include in Your Request

For **corrections**:

| Include                | Example                             |
| ---------------------- | ----------------------------------- |
| Page URL or name       | "Orbiting technique page"           |
| What's wrong           | "The execution steps are incorrect" |
| Correct information    | "Step 2 should say..."              |
| Source (if applicable) | How you know it's correct           |

For **new content**:

| Include         | Example                        |
| --------------- | ------------------------------ |
| Topic           | "Advanced stealing techniques" |
| Why it's needed | "Many players ask about this"  |
| Information     | What should be covered         |
| Your expertise  | How you know this              |

---

## Types of Requests

### :material-pencil: Content Corrections

- Factual errors
- Typos and grammar
- Outdated information
- Broken links

### :material-plus: New Content

- Missing techniques
- Map guides
- Server documentation
- Strategy guides

### :material-image: Media Requests

- Screenshots needed
- Video demonstrations
- Diagrams and illustrations

### :material-translate: Translations

- Request language support
- Offer translation help

---

## Request Template

Use this template when submitting:

```
**Type of Request:** [Correction / New Content / Other]

**Page:** [Which page, or "New page needed"]

**Description:**
[Detailed description of the change]

**Reason:**
[Why this change should be made]

**Additional Information:**
[Screenshots, examples, sources]
```

---

## What Happens Next

1. :material-eye: **Review** - A maintainer reviews your request
2. :material-message: **Discussion** - We may ask for clarification
3. :material-pencil: **Implementation** - Changes are made
4. :material-check: **Published** - Updates go live

Response times vary but we aim to acknowledge requests within a few days.
